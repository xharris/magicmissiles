- This is a game made with Godot v4.6.stable.official [89cea1439]

## Logging

- Check the logs. Logs are written to `user://logs/godot.log`
  - Windows: `%APPDATA%\Godot\app_userdata\magicmissiles`
  - macOS: `~/Library/Application Support/Godot/app_userdata/magicmissiles`
  - Linux: `~/.local/share/godot/app_userdata/magicmissiles`
- Log lines that only contain `...` mean that it is a repeat of the previous line
